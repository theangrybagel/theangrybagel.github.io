Weapon Classes

Understanding weapon classes

A weapon class (when given to a ship) is determined by going through each class of weaponry, determining if it can be used on the ship, and labeling the number of weapons of a specific type a ship can have followed by the class name. For instance, if a ship can hold two W class weapons, the weapon class would be 2W. If it can hold two W class weapons, a B class weapon, and 6 F class weapons, the weapon class would be 6F2W1B. If it is capable of holding two B class weapons and 2 W class weapons, it can be represented as 2B2W or just 2BW.

Class B weaponry

Weaponry mounted on the bottom of a ship.

Class F weaponry

Weaponry mounted on the front of a ship.

Class C weaponry

Custom weaponry.

Class R weaponry

Rear mounted weaponry.

Class W weaponry

Mounted on the wings of a ship.
